# 🦕💰 น้องออม — Saving & Investment Planner

ตัวช่วยวางแผนการเงิน-การลงทุนสำหรับพนักงานออฟฟิศ ในรูปแบบ Interactive HTML Dashboard ไฟล์เดียว

มาสคอตคือ **น้องออม** ไดโนเสาร์เศรษฐีหมวกสูง 🎩 ที่จะช่วยให้เรื่องเงินดูง่ายและน่าใช้

## ✨ สิ่งที่มีในแดชบอร์ด

### 💰 จัดสรรเงินเดือน 50/30/20
- แบ่ง Needs / Wants / Savings เป็นบาทจริง
- เช็คสุขภาพการเงิน: DSR, เงินสำรองฉุกเฉิน, % เก็บได้
- ตารางตัวอย่างหมวดย่อยพร้อม icon น่ารัก

### 📈 จัดพอร์ตลงทุน
- สัดส่วน หุ้น / ตราสารหนี้ / ทอง / คริปโต / เงินสด
- ปรับตามความเสี่ยง + กฎ "100 ลบอายุ"
- ถ้าเงินสำรองยังไม่ครบ ระบบจะลดเงินลงทุนอัตโนมัติ

### 🧾 ภาษี & สิทธิลดหย่อน
- คำนวณภาษีตามอัตราขั้นบันได 2568
- แนะนำการใช้สิทธิ SSF / RMF / ThaiESG / PVD ตามฐานภาษี
- โชว์ "ประหยัดภาษีได้" เป็นบาทชัดๆ

### 🎯 เป้าหมายระยะยาว
- เกษียณ: คำนวณเงินก้อนที่ต้องมี (ปรับเงินเฟ้อ 3%) + DCA เดือนละเท่าไหร่
- บ้าน/ดาวน์รถ: เก็บเดือนละเท่าไหร่
- การศึกษาลูก: ค่าเรียนตอนลูกเข้ามหา'ลัย (เงินเฟ้อ 5%/ปี)

### 📰 ข่าว & ราคาตลาด (Live)
- Tip การลงทุนหมุนเวียนรายวัน (20 tips)
- TradingView ticker tape: SET, ทอง, USD/THB, BTC, S&P500
- ราคาสด: ทองคำโลก, USD/THB, BTC, ETH (CoinGecko + ECB)
- ข่าวจาก RSS: กรุงเทพธุรกิจ, ฐานเศรษฐกิจ, THE STANDARD Wealth, Bloomberg, Investing.com
- ลิงก์รวม 12 แหล่งข้อมูลที่เชื่อถือได้

## 🚀 วิธีใช้

เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ใดก็ได้ — ไม่ต้องติดตั้งอะไรเพิ่ม

```bash
open index.html
```

หรือเปิดผ่าน GitHub Pages: https://chochew.github.io/saving-project/

## 🎨 เทคนิคที่ใช้

- HTML / CSS / JavaScript ล้วน (ไม่มี framework)
- [Chart.js](https://www.chartjs.org/) สำหรับ donut chart
- Google Fonts: Mali (ฟอนต์ไทยน่ารัก)
- SVG inline icons วาดเองทุกตัว
- APIs ฟรี ไม่ต้องใช้ key:
  - [Frankfurter](https://www.frankfurter.app/) (forex)
  - [CoinGecko](https://www.coingecko.com/) (crypto + gold)
  - [rss2json](https://rss2json.com/) (news feeds)
  - [TradingView](https://www.tradingview.com/) widget (ticker tape)
